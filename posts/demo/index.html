Hugo + org 使用

emacs的org mode 加上ox-hugo可以方便的生产blog。

把org文件导出成md

使用 ox-hugo 把 org 文件导出为 md 文件

方式一：手动导出

C-c C-e H H

方式二：保存 org 文件时自动导出

在站点根目录新建 dir-locals.el 文件

vi .dir-locals.el
(("content-org/"
  . ((org-mode . ((eval . (org-hugo-auto-export-mode)))))))

启动 hugo server

编译hugo

在hugo的根目录下，执行以下命令

hugo server -D

把生成好的public目录通过git传到github

初始化git仓库

cd public
git init
git remote add origin git@github.com:vritser/vritser.github.io.git

提交，推送

cd public
git init
git remote add origin git@github.com:vritser/vritser.github.io.git
